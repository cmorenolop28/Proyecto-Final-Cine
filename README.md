# 🎬 Sistema de Administración de Cine

## 📌 Descripción
Sistema web desarrollado para la gestión de un cine. Permite administrar películas, salas, funciones y la venta de boletos con selección de asientos.

---

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP
